Часть 1

  5. 1.5 Выберите верное равенство:
    • \(\dfrac{a}{c} + \dfrac{b}{c} = \dfrac{a+b}{c}\)
    • \(\dfrac{a}{c} + \dfrac{b}{c} = \dfrac{ab}{c}\)
    • \(\dfrac{a}{c} + \dfrac{b}{c} = \dfrac{a}{c} + b\)
    • \(\dfrac{a}{c} + \dfrac{b}{c} = \dfrac{a+b}{2c}\)
  6. 1.6 Сложите дроби: \(\dfrac{3x}{7} + \dfrac{2x}{7}\).
  7. 1.7 Выполните вычитание: \(\dfrac{5a}{b} - \dfrac{2a}{b}\).
