Основные определения и табличные значения

Синус острого угла \(\sin\alpha = \dfrac{a}{c}\)
Косинус острого угла \(\cos\alpha = \dfrac{b}{c}\)
Тангенс острого угла \(\operatorname{tg}\alpha = \dfrac{a}{b}\)
Котангенс острого угла \(\operatorname{ctg}\alpha = \dfrac{b}{a}\)

Часть 1 Базовые задачи
  1. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) катет \(BC = 5\), катет \(AC = 12\). Найдите:
    • \(\sin A\)
    • \(\cos A\)
    • \(\operatorname{tg} A\)
    • \(\operatorname{ctg} A\)
  2. В прямоугольном треугольнике \(MNP\) (\(\angle N = 90^\circ\)) гипотенуза \(MP = 17\), катет \(MN = 8\). Найдите:
    • \(\cos P\)
    • \(\sin M\)
    • \(\operatorname{tg} P\)
    • \(\operatorname{ctg} M\)
  3. Вычислите:
    • \(\sin30^\circ + \cos60^\circ\)
    • \(\operatorname{tg}45^\circ \cdot \operatorname{ctg}45^\circ\)
    • \(\sin^2 30^\circ + \cos^2 30^\circ\)
    • \(\operatorname{tg}30^\circ \cdot \operatorname{ctg}30^\circ\)
  4. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) \(\cos A = 0,8\). Найдите \(\sin A\).
Часть 2 Применение определений
  1. Известно, что \(\sin\alpha = \frac{5}{13}\), где \(\alpha\) — острый угол. Найдите \(\cos\alpha,\ \operatorname{tg}\alpha,\ \operatorname{ctg}\alpha\).
  2. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) гипотенуза \(AB = 10\), \(\sin A = 0,6\). Найдите катеты \(BC\) и \(AC\).
  3. В треугольнике \(DEF\) (\(\angle E = 90^\circ\)) катет \(DE = 2\sqrt{3}\), гипотенуза \(DF = 4\). Найдите \(\angle F\) и \(\angle D\) (в градусах).
  4. Сравните значения выражений:
    • \(\sin20^\circ\) и \(\cos70^\circ\)
    • \(\operatorname{tg}40^\circ\) и \(\operatorname{ctg}50^\circ\)
Часть 3 Вычисления и соотношения
  1. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) катет \(AC = 8\), \(\cos A = 0,6\). Найдите гипотенузу \(AB\) и катет \(BC\).
  2. В равнобедренном прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) катет \(AC = 6\). Найдите синус и косинус угла \(A\).
  3. Дано: \(\operatorname{tg}\alpha = \frac{3}{4}\). Найдите \(\sin\alpha\) и \(\cos\alpha\) (угол \(\alpha\) острый).
  4. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) \(\angle A = 30^\circ\), катет \(BC = 5\). Найдите гипотенузу \(AB\) и катет \(AC\), а затем \(\sin A\), \(\cos A\), \(\operatorname{tg} A\), \(\operatorname{ctg} A\).
Часть 4 Геометрические приложения
  1. В треугольнике \(ABC\) (\(\angle C = 90^\circ\)) \(AB = 13\), \(BC = 5\). Найдите \(\sin A\), \(\cos A\), \(\operatorname{tg} A\), \(\operatorname{ctg} A\).
  2. Используя равносторонний треугольник со стороной \(2\), найдите \(\sin60^\circ,\ \cos60^\circ,\ \operatorname{tg}60^\circ,\ \operatorname{ctg}60^\circ\).
  3. В равнобедренном треугольнике \(ABC\) с основанием \(AC = 16\) и боковой стороной \(AB = 10\) найдите \(\operatorname{tg} \angle BAC\).
  4. В прямоугольной трапеции \(ABCD\) (\(\angle A = \angle B = 90^\circ\)) основания \(BC = 4\), \(AD = 10\), боковая сторона \(CD = 8\). Найдите \(\sin \angle D\).
Часть 5 Доказательства и обобщения
  1. Известно, что \(\operatorname{ctg}\alpha = 2 + \sqrt{3}\). Найдите \(\sin\alpha\) и \(\cos\alpha\) (угол \(\alpha\) острый).
  2. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) гипотенуза \(AB = 12\), \(\angle A = 30^\circ\). Найдите катет \(BC\) и затем \(\sin A,\ \cos A,\ \operatorname{tg} A,\ \operatorname{ctg} A\).
  3. Докажите, что для любого острого угла \(\alpha\) справедливо неравенство \(\sin\alpha + \cos\alpha > 1\) (используйте определения и неравенство треугольника).
  4. Докажите, что для любого острого угла \(\alpha\) \(\sin\alpha = \cos(90^\circ-\alpha)\) (используйте определения в прямоугольном треугольнике).
✦ Проверка знаний
  1. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) \(BC = 7\), \(AC = 24\). Найдите \(\sin A\), \(\cos A\), \(\operatorname{tg} A\), \(\operatorname{ctg} A\).
  2. Вычислите: \(\sin45^\circ \cdot \cos45^\circ + \operatorname{tg}30^\circ \cdot \operatorname{ctg}30^\circ\).
  3. Найдите \(\operatorname{tg}\alpha\), если \(\cos\alpha = 0,2\) (угол \(\alpha\) острый).
  4. В прямоугольном треугольнике \(MKN\) (\(\angle K = 90^\circ\)) гипотенуза \(MN = 13\), \(\sin M = \frac{5}{13}\). Найдите катеты.
  5. В равнобедренном прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) катет \(AC = 5\). Найдите синус и косинус угла \(A\).
  6. Дано: \(\operatorname{ctg}\alpha = \frac{5}{12}\). Найдите \(\sin\alpha\).
  7. В прямоугольном треугольнике \(ABC\) (\(\angle C = 90^\circ\)) высота \(CH = 3\), проекция катета \(AC\) на гипотенузу равна 4. Найдите \(\cos A\).
  8. В прямоугольной трапеции \(ABCD\) (\(\angle A = \angle B = 90^\circ\)) основания \(BC = 6\), \(AD = 12\), боковая сторона \(CD = 10\). Найдите \(\sin \angle D\).
  9. Сравните \(\sin20^\circ\) и \(\cos80^\circ\).
  10. Докажите, что для любого острого угла \(\alpha\) \(\sin\alpha = \cos(90^\circ-\alpha)\).
